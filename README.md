# ESA_Statistics_Workshop_2025

For each part of the analysis, data sets (after some processing) are provided in the 'data' folder.
(1) Partitioning beta-diversity into turnover and nestedness components: ladybeetle_betadiversity.csv
(2) Nonmetric multidimensional scaling analysis: ladybeetle_data.csv and landscape_data.csv
(3) Generalized additive models: ladybeetle_gams.csv

All analysis code for each part of the analysis is provided in the R script 'LB_analyses'. 

Workshop materials based on:
Perry, KI, CA Bahlai, TJ Assal, CB Riley, KJ Turo, L Taylor, J Radl, YA Delgado de la flor, FS Sivakoff, and MM Gardiner. 2024. Landscape change and alien invasions drive shifts in native lady beetle communities over a century, Ecological Applications, 34(7): e3024.
https://doi-org.proxy.lib.ohio-state.edu/10.1002/eap.3024

The full analysis code and development history are available in the GitHub repository: https://github.com/BahlaiLab/Ohio_ladybeetles
